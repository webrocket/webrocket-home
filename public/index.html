Real-time Web for masses

What is real-time Web? It's a bi-directional communication between browsers and web applications, usually solved by implementing WebSockets server. But WebSockets are complicated and hard to scale! You don't need to care about it anymore. WebRocket is a broker - a tie between very efficient WebSockets server implementation and your backend apps.

WebRocket combines very efficient and scalable WebSockets server with internal messaging and asynchronous backend communication. Core system is written in Go programming language - reliable and extremally fun language, which we believe will rock development of distributed apps in the nearest future.

No language barrier

WebRocket backends can be implemented in any programming language, which means that you don't have to learn new languages or complicated technologies to use the power of WebSockets in your apps. You can spend your saved time on learning Klingon instead... :)

Open source platform

Main motivation behind WebRocket was a need to create an open platform at top of WebSockets which will make building real-time Web applications faster and easier, and more fun. Our code will never be hidden, we want to bring real-time Web to everyone!
